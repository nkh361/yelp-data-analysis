# Data-Driven Insights into Restaurant Markets Using Hybrid Clustering and Regression on Yelp Data

### Build Instructions for Flask Demo
Clone the repository: 
```bash
git clone https://github.com/nkh361/yelp-data-analysis.git
cd yelp-data-analysis
```

Utilize ```build.sh``` to load and run the Flask application:
```bash
chmod +x build.sh
./build.sh
```

---

### Requirements
This project utilizes Python virtual environments. Dependencies are installed automatically by ```build.sh```
using the included ```requirements.txt```.
